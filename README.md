# Complaint Portal
A basic CRUD app build using Flask.

**Requirements**
- Flask==1.0.2
- Flask-Bootstrap==3.3.7.1
- Flask-Moment==0.6.0
- Flask-Script==2.0.6
- Flask-SQLAlchemy==2.3.2
- Flask-WTF==0.14.2
- SQLAlchemy==1.2.11
- WTForms==2.2.1

# Setup

Make sure all dependencies are installed in your virtual environment. 

create all dbs

run inex.py to start the server
